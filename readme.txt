Keep track of Users login and logout time even if user closes the browser without logging out.

== Description ==

This plugin will keep the track of Users login and logout time. Track is maintained even if the user has closed the browser without logging out from their account. Admin can check the IP address and history of the user as well. You can check it under Dashboard.

== Installation ==

1. Upload the plugin rnd-active-login files to the `/wp-content/plugins/` directory
2. Activate the RND Active Login plugin through the 'Plugins' menu in WordPress.
3. Goto Dashboard.
